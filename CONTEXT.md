# Sagi — Полный контекст проекта

## Что такое Sagi

Sagi Community — мобильное приложение (iOS + Android) и веб-платформа для управления сообществами. Первый клиент — клуб лояльности МФЦА (Международный финансовый центр Астаны).

---

## Стек

- React 19 + TypeScript
- Tailwind CSS v4
- Vite
- lucide-react (иконки)
- react-router v7

---

## Три роли в приложении

### 1. Для сообщества (администратор)
- Верификация участников
- Аналитика в реальном времени
- Контроль сроков членства
- Управление партнёрской сетью
- Бейдж: `bg-[#EDFAF3] text-[#1E9E5A]`

### 2. Для участника (резидент)
- Apple Wallet карта (цифровое членство)
- Карта лояльности с QR ID
- Каталог скидок и офферов
- Лента событий и новостей
- Verified Networking (обмен контактами с взаимным согласием)
- Бейдж: `bg-[#EEF2FF] text-[#4338CA]`

### 3. Для бизнеса (партнёр)
- Сканирование QR-карт участников
- Управление офферами и скидками
- Статистика визитов
- Персональный менеджер Sagi
- Бейдж: `bg-[#FFF7ED] text-[#C2410C]`

## Коммерческая модель

- **Setup Fee**: $2 500 (единоразово — внедрение и оцифровка базы)
- **Ежемесячно**: $20 за участника в месяц
- **Ежегодно**: $170 за участника в год (скидка 30%, экономия $70)
- **Окупаемость**: 15 продлений при стоимости членского взноса $170
- **Киллер-аргумент**: система автоматически блокирует доступ просроченных карт, исключая использование привилегий неплательщиками

## SagiLanding (`/landing`) — B2B лендинг

Светлый дизайн. Для партнёров и инвесторов.

**Секции по порядку:**
1. **NAV** — логотип "sagi" + бейдж "Community · МФЦА" + ссылки (Возможности, Для бизнеса, Тарифы)
2. **HERO** — заголовок "От PDF-каталога к живой экосистеме привилегий" + кнопка "Посмотреть тарифы" + статы (12 функций / $20 / 30%) + анимированный телефон с картой AIFC MEMBER
3. **APP PREVIEW** — 3 iPhone фрейма с ролями (screen1/2/3)
4. **12 ФУНКЦИЙ** — сетка 3 колонки, Apple Wallet выделен зелёной карточкой, Контроль сроков span-2, Менеджер — отдельная широкая карточка
5. **ИНСТРУМЕНТЫ** — 2 колонки: Панель управления (admin) + Кабинет бизнеса
6. **ТАРИФЫ** — 2 карточки цен ($20 и $170) + общий блок с фичами по группам
7. **ROI** — 3 причины почему выгодно
8. **FOOTER CTA** — тёмная карточка с призывом

## AppStoreLanding (`/app`) — для App Store

Светлый дизайн. Для подачи в Apple App Store / Google Play.

**Обязательные элементы (требования Apple):**
- Privacy Policy секция с якорем `#privacy`
- Support URL: `mailto:support@sagi.kz`
- Privacy contact: `privacy@sagi.kz`

**Секции:**
1. NAV — логотип + ссылки "Конфиденциальность" и "Поддержка"
2. HERO — иконка приложения + название + кнопки App Store / Google Play + 5 звёзд
3. 3 РОЛИ — скриншоты с описаниями
4. ФУНКЦИИ — 9 карточек
5. DOWNLOAD CTA — повтор кнопок скачивания
6. PRIVACY POLICY — полный текст политики конфиденциальности
7. FOOTER

**В App Store Connect вставить:**
- Marketing URL → `https://домен/app`
- Support URL → `https://домен/app`
- Privacy Policy URL → `https://домен/app#privacy`

## Предпочтения по коду

- Без framer-motion (не установлен), анимации через CSS
- Иконки только из lucide-react
- Tailwind inline-стили только когда класс не покрывает нужное значение
